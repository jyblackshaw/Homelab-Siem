# Active Directory Setup

Documentation of the Domain Controller configuration, organizational unit structure, Group Policy Objects, and domain-joining Linux endpoints for centralized identity management.

## Sections to Cover
- Domain Controller installation and domain creation
- OU structure (SOC Team, IT Admins, Standard Users)
- Security groups and role-based access
- GPOs: password policy, account lockout, restricted local admin, audit logging
- Domain-joining Linux hosts via realmd + SSSD
- Audit log forwarding to Splunk
